## Download and install FusionAuth

**Objectives**

To set up this workspace, the following steps are followed:

# 1. Install FusionAuth

```
/usr/bin/bash install.sh
```

# 2. Start FusionAuth

Start up the service:

```
fusionauth/bin/startup.sh && /usr/bin/bash wait.sh
```
